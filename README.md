# Cab Booking System – SQL Data Analysis Project

This project demonstrates a Cab Booking System designed for data analysis using MySQL. 
It focuses on managing customers, drivers, bookings, trips, and revenue to generate meaningful business insights.

## Tools Used
- MySQL
- SQL (DDL, DML, Joins, Subqueries, Aggregations)

## Project Features
- Customer, Driver, Cab, Booking, Trip, and Feedback management
- CRUD operations on multiple related tables
- Advanced SQL queries for analysis

## Key Analysis Performed
- Most frequent customers and cancellation analysis
- Driver performance and rating analysis
- Revenue trends and route analysis
- Vehicle-type based revenue comparison

## Key Insights
- Identified high-value customers and frequent cancellations
- Evaluated driver reliability using ratings and cancellations
- Analyzed revenue patterns and peak booking behavior
