# Cab Booking System SQL Project

This project demonstrates database design and SQL analysis for a cab booking platform.

## Database Tables
- Customers
- Drivers
- Cabs
- Bookings
- Trips
- Feedback

## Key Analysis
- Identify top customers by number of bookings
- Analyze driver performance and ratings
- Track revenue and booking trends
- Identify high demand routes

## Tools Used
- MySQL
- SQL
